# Title

An application to display the group list and how they prefer their coffee.
Was applied by Flutter and Firebase(Auth & Firestore).

## Overview

The App Has:
- Login Interface
- Signup Interface
- Loading Interface
- Home
- Update sheet

Coffee attributes: (person name, sugars for his coffee , coffee strength)

**Note: you need to link the application with your database on Firebase (google-services/info) to work well with you.**
